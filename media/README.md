# Media Folder

Place your media files here:

- **photo.jpg** - Photo for Polaroid page (page 17)
- **video.mp4** - Birthday video for page 23

## Usage

The website references media files from this folder:
- Polaroid uses `media/photo.jpg`
- Video player uses `media/video.mp4`

Update `config.json` to change file paths if needed.
